Lab #4: Paging Lab (Java)

To compile: javac PagingLab.java
To run: java PagingLab [5 arguments in this order: machine size (int), page size (int), process size (int), job mix (int), number of references for each process (int), lower case replacement algorithm either lru, lifo, or random (String)]

e.g.: java PagingLab 10 10 20 1 10 lru

There is only one class file and everything is handled within it. The program will print the necessary output.
